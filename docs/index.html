关于 OneDocs

OneDocs 是一款开源的文件分析工具，可以对输入的各种文本文件进行析读，并摘选出对用户有用的信息内容。

OneDocs 适用于处理内容比较繁杂且有大量无关内容的文件，可以轻松地从中采撷重点信息，最终整理并渲染展示，用户亦可复制全文或导出备份。
